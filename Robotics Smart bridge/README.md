# Smart Bridge System

## Overview
Smart Bridge is an automated flood detection and safety system that prevents vehicles from crossing unsafe water levels using sensors and actuators. :contentReference[oaicite:2]{index=2}

---

## Features
- Water level detection
- Automatic barrier control (servo motor)
- LED warning system
- Threshold-based automation

---

##Concepts
- Automation systems
- Sensor-based decision making
- Embedded programming
- Safety systems design


## Components
- Arduino Uno
- Water Level Sensor
- Servo Motor
- LEDs
- Power Supply


## Working
- Sensor reads water level
- If level exceeds threshold:
  - Barrier closes
  - Warning lights activate

- Flood-prone areas
