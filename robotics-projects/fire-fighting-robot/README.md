# Fire Fighting Robot

## Overview
The Fire Fighting Robot is an autonomous robotic system designed to detect and extinguish fire using sensors and actuators. It operates using the Input → Process → Output (IPO) model and demonstrates real-world disaster-response robotics.

This project is inspired by industrial firefighting robots like Thermite RS3. :contentReference[oaicite:0]{index=0}

---

## Features
- Flame detection using flame sensors
- Autonomous navigation using DC motors
- Water pump control using relay module
- Servo-based directional spraying
- Closed-loop control system

---

## Concepts Used
- Robotics (Sense → Decide → Act)
- IPO Model
- Embedded Systems Programming
- Sensor Integration
- Conditional Logic

---

## Components Required
- Arduino Uno
- Flame Sensors
- Motor Driver Shield (L298N / similar)
- DC Motors & Chassis
- Servo Motor
- Relay Module
- Water Pump
- Battery Pack

---

## Working Principle
1. Sensors detect fire
2. Microcontroller processes input
3. Robot moves toward fire
4. Pump activates to extinguish fire

---

## Applications
- Industrial safety systems
- Disaster response robots
- Smart fire detection systems
